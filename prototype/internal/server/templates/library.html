About Library

Documentation Collections: Pull documentation from URLs, local files, or git repositories to provide context for AI agents.

Include Modes:

  • Auto: Include when file paths match patterns
  • Explicit: Include only with --library flag
  • Always: Always include in prompts

Storage: Collections can be stored per-project or shared globally across all projects.

CLI Usage: 

mehr library pull https://docs.example.com/
mehr library list
mehr implement --library react-docs
