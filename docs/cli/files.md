# kvelmo files

File browser.

## Usage

```bash
kvelmo files [path]
```

## Examples

```bash
# List current directory
kvelmo files

# List specific path
kvelmo files src/

# Show file contents
kvelmo files src/main.go
```

## What It Does

- Lists directory contents
- Shows file contents
- Provides quick navigation
