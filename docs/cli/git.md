# kvelmo git

Git operations.

## Usage

```bash
kvelmo git <subcommand>
```

## Subcommands

| Command | Description |
|---------|-------------|
| `status` | Show git status |
| `diff` | Show changes |
| `branch` | Show current branch |

## Examples

```bash
# Status
kvelmo git status

# Diff
kvelmo git diff

# Branch
kvelmo git branch
```

## Notes

For most git operations, use `git` directly. These commands are convenience wrappers.
