# kvelmo projects

Project registry management.

## Usage

```bash
kvelmo projects
```

## Output

```
Registered projects:
  /home/user/project-a (active)
  /home/user/project-b
  /home/user/project-c
```

## What It Shows

- Project paths
- Active status
