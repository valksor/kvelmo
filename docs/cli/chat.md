# kvelmo chat

Interactive chat with the agent.

## Usage

```bash
kvelmo chat
```

## What It Does

Opens an interactive chat session with the configured agent.

## Examples

```bash
# Start chat
kvelmo chat

# In chat:
> How should I structure the authentication?
> What files need to change?
> Exit
```

## When to Use

- Need to ask questions about the task
- Want guidance before implementing
- Exploring approaches
