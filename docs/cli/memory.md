# kvelmo memory

Memory management commands.

## Usage

```bash
kvelmo memory <subcommand>
```

## Subcommands

| Command | Description |
|---------|-------------|
| `index` | Index the codebase |
| `search <query>` | Search memory |
| `clear` | Clear all memory |

## Examples

```bash
# Index codebase
kvelmo memory index

# Search
kvelmo memory search "authentication"

# Clear
kvelmo memory clear
```
