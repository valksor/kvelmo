About Stacked Features

Stacked features allow you to work on Feature B while Feature A is waiting on code review.

  • Create a dependent feature: mehr start feature-b --depends-on feature-a
  • Sync PR status: Click "Sync PR Status" to fetch the latest PR states
  • Rebase after parent merges: When a parent PR merges, children are marked "needs-rebase". Click "Rebase All" to rebase them

Use mehr stack in the CLI for more options including --graph and --mermaid visualization.
